Snake Game 🐍

A classic Snake Game built using HTML, CSS, and JavaScript.
The player controls the snake using keyboard arrow keys to eat food, grow longer, and increase the score while avoiding collisions.

📌 Features

✔ Classic Snake Game mechanics<br>
✔ Keyboard controls (Arrow keys)<br>
✔ Random food generation<br>
✔ Score tracking<br>
✔ Smooth gameplay using JavaScript Canvas

🛠 Technologies Used

HTML5 – Game structure

CSS3 – Styling

JavaScript (ES6) – Game logic

Canvas API – Rendering the game board

📂 Project Structure
snake-game<br>
├── index.html<br>
├── style.css<br>

🎮 How to Play

Open the game in your browser.

Use Arrow Keys to move the snake:

⬅ Left

➡ Right

⬆ Up

⬇ Down

Eat the red food to increase your score.

Avoid hitting the walls or your own body.

🚀 How to Run the Project

Clone the repository

git clone https://github.com/Rup-Ali/-Frontend-Developer-Portfolio/tree/main/Snake.Game

Open the project folder

Run the game by opening: index.html in your browser.
